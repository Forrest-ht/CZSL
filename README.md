# Vision-Guided Prototype Expansion and Mutual Contrast for Compositional Zero-Shot Learning

Official repository for the paper  
**Vision-Guided Prototype Expansion and Mutual Contrast for Compositional Zero-Shot Learning**

## Highlights

- tackles **prototype sparsity** in compositional zero-shot learning
- mitigates **projected domain drift** with mutual contrastive learning
- introduces a **Vision-guided Explanation Matching Module (VEMM)** for attribute prototype expansion
